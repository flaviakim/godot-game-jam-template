# What files to put here

- all files created in external editor like Asprite or Audio Software
- e.g. .png, .jpg or .wav, .mp3...

please organize in sub folders as follows, feel free to add more subfolders (e.g. for levels or different menus)
```
- audio
    | - sfx...
    | - music...
- graphics
    | - ui
    | - logos
    | - game
```
